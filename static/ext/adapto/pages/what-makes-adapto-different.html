How is Adapto Different?

Adapto is no ordinary theme. Here are the some of the stand-out features packed into Adapto.

Seriously Adaptive

If you're haven't checked out the style adaptation widget (to the left), go ahead and give it a go. Adapto can be seriously tweaked to meet you're particular tastes and needs. Moreover, it has a memory. That's right, go ahead and change pages, close the window and come back, even turnoff your computer... Adapto will remember your adaptation. Need to reset the options? Simply click either of the two reset buttons.

Here are some of the customizations possible with the adaptation widget:

  • Light/Dark Color Schemes
  • 20 Primary Color Schemes
  • Background Noise Toggle for:
    • Main Site Body
    • Navigation Header
    • Page Header
    • Footer
  • Extra Header with Content Options:
    • Store
    • Account
    • Social Media (2 options)
    • Phone
  • Wide/Boxed Option for two areas:
    • Header & Footer
    • Body
  • Top/Bottom Space Option
  • Background Colors and Textures/Patterns for Boxed Option

Built with Blocks

What's a block? In Adapto terms, think "reusable chunk of code for building my own site." This approach towards architecting Adapto helps us reach the widest audience possible without sacrificing anything. Whether you're a seasoned developer or a curious newbie, the theme's simple and intuitive blocks will enable you to fully adapt Adapto to your needs.

Built with Sass

SASS is a CSS pre-processor. If you're already in the know, you already know. If not, the next few lines might just change your life (or at least the way you work with CSS).

First off, there's absolutely nothing to be afraid of with Sass. Why's this? Because you can write plain-ole' css in a .scss file and everything works great (.scss files are one of the two types of Sass files... the other's a .sass file and it's rather sweet, but it's a bit beyond the current scope). The only catch is that you have to, well, process sass files (hence the pre in pre-processor - but there are some great (and free) tools available to easily do this for you.

The best part

learning even a little bit of Sass tricks can dramatically enhance your developer abilities. Even if you're not a developer (hey, all sorts of folks make websites, right?), learning just the basics of Sass will make you feel like a CSS superhero (to a degree, at least :)

Adapto & Sass

Adapto is fully built with Sass. We've used the official Sass port of Bootstrap 3.1 to bring you all the latest and greatest that both Bootstrap and Sass have when their powers are combined.

What does this mean?

This means that upon purchasing Adapto you'll open your preferred theme variables file and quickly and easily begin adapting customizing Adapto for it to fully meet your needs. It's quick, easy, and even quick enjoyable.

I don't know Sass, what to do?

There's no shortage of Sass tutorial material available. Here's what we recommend:

  • Read this short article to gain a general understanding of Sass.
  • Head over to Codepen and create a pen to begin experimenting
  • Decide how you'd like to process your Sass into CSS, using either:
    • Command line
    • Desktop Application
  • For both approaches, checkout the official getting started guide.
  • If you prefer video, here's a great series of tutorials on getting setup and working with Sass: Sass Tutorial - Level Up Tuts
Now What?

Go have fun :)

The powers of Sass are incredible. They've helped us create a theme that's ultra-adaptable via some simple magic called variables, if statements, and other otherwise programming logic into the world of CSS. If you don't want to use Sass for some reason, don't worry... the theme of course still includes all the compiled CSS files.
